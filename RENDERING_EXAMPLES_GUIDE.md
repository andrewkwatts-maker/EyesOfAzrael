# Rendering Examples Guide

This document explains the rendering examples generated by the validation script.

**File:** `rendering-examples.json`
**Purpose:** Demonstrate how each type of asset renders in all 5 display modes

---

## Display Modes Explained

### 1. PAGE Mode
**Use Case:** Full dedicated page for an entity
**Required Fields:** name, description
**Example URL:** `/mythos/greek/deities/zeus`

**Template:**
```html
<div class="entity-page">
  <h1>{name}</h1>
  <div class="icon">{icon}</div>
  <div class="metadata">
    <span class="mythology">{mythology}</span>
    <span class="type">{type}</span>
  </div>
  <div class="description">{description}</div>
</div>
```

**Features:**
- Complete content display
- Full description
- All metadata visible
- Icon prominently displayed
- Primary navigation target

---

### 2. PANEL Mode
**Use Case:** Side panel, modal, or drawer view
**Required Fields:** name, description
**Example Context:** Quick reference panel, entity preview

**Template:**
```html
<div class="entity-panel">
  <h3>{name}</h3>
  <p>{description (truncated to 200 chars)}</p>
</div>
```

**Features:**
- Compact display
- Truncated description
- Suitable for overlays
- Quick reference format

---

### 3. SECTION Mode
**Use Case:** Section within a larger page
**Required Fields:** name, description
**Example Context:** Embedded in comparison page, mythology overview

**Template:**
```html
<section class="entity-section">
  <h2>{name}</h2>
  <p>{description}</p>
</section>
```

**Features:**
- Semantic HTML
- Full description
- Part of larger document
- Proper heading hierarchy

---

### 4. LINK Mode
**Use Case:** Simple hyperlink reference
**Required Fields:** name
**Example Context:** Navigation menus, related entities lists

**Template:**
```html
<a href="/{collection}/{id}" class="entity-link">{name}</a>
```

**Features:**
- Minimal display
- Navigation element
- Can work without description
- Most flexible mode

---

### 5. PARAGRAPH Mode
**Use Case:** Inline text reference
**Required Fields:** name, description
**Example Context:** Within narrative text, inline definitions

**Template:**
```html
<p class="entity-paragraph">
  <strong>{name}:</strong> {description (truncated to 150 chars)}
</p>
```

**Features:**
- Inline format
- Brief description
- Context-appropriate
- Flows with text

## Using the Examples

### 1. View All Examples
```javascript
// Load the examples file
const examples = require('./rendering-examples.json');

// Get all collections
console.log(Object.keys(examples));

// View specific collection example
console.log(examples.deities);
```

### 2. Test Rendering Modes
```javascript
// Get an example
const deity = examples.deities;

// Access specific rendering mode
console.log(deity.modes.page.html);
console.log(deity.modes.panel.html);

// Check if can render
console.log(deity.modes.page.canRender); // true/false
```

### 3. Implement in Your Renderer
```javascript
function renderEntity(entity, mode) {
  switch(mode) {
    case 'page':
      return renderPageMode(entity);
    case 'panel':
      return renderPanelMode(entity);
    case 'section':
      return renderSectionMode(entity);
    case 'link':
      return renderLinkMode(entity);
    case 'paragraph':
      return renderParagraphMode(entity);
  }
}
```

## Best Practices

### 1. Always Check canRender
```javascript
if (example.modes.page.canRender) {
  // Safe to render
  return example.modes.page.html;
} else {
  // Fallback to link mode
  return example.modes.link.html;
}
```

### 2. Graceful Degradation
```javascript
// Preferred order: page -> panel -> section -> paragraph -> link
const modes = ['page', 'panel', 'section', 'paragraph', 'link'];
for (const mode of modes) {
  if (entity.modes[mode].canRender) {
    return entity.modes[mode].html;
  }
}
```

### 3. Mode Selection by Context
```javascript
function selectRenderMode(context) {
  switch(context) {
    case 'main-content':
      return 'page';
    case 'sidebar':
      return 'panel';
    case 'embedded':
      return 'section';
    case 'inline':
      return 'paragraph';
    case 'navigation':
      return 'link';
  }
}
```
