How It Works

The metadata integration system enhances corpus search in three ways:

  1. Alternate Name Expansion: Searches for "Enki" also find "Ea" and "Nudimmud"
  2. Cross-Cultural Equivalents: Searches for "Zeus" also find "Jupiter" references
  3. Entity Metadata Enrichment: Results show entity information, mythology, and related names
