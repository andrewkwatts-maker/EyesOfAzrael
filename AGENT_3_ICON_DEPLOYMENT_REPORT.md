## ğŸ“ Usage Guide

### For Developers

**Running the Deployment Script:**

```bash
# Preview changes (dry-run)
node scripts/inject-icons-to-html.js --dry-run

# Apply changes to all files
node scripts/inject-icons-to-html.js --execute

# Process a single file
node scripts/inject-icons-to-html.js --file mythos/greek/deities/zeus.html --execute
```

**Adding New Icons:**

1. Create SVG in appropriate `/icons/` subdirectory
2. Add entry to `/icons/icon-type-registry.json`:
```json
{
  "icon-name": {
    "path": "icons/category/icon-name.svg",
    "keywords": ["keyword1", "keyword2"],
    "useCases": ["Use case description"]
  }
}
```
3. Run deployment script with `--execute`

**Using Icons in HTML:**

```html
<!-- Basic usage -->
<img src="/icons/deity-icon.svg" alt="Deity" class="entity-icon">

<!-- With size class -->
<img src="/icons/deity-domains/war.svg" alt="War" class="entity-icon hero-size">

<!-- With container -->
<div class="entity-icon-container">
  <img src="/icons/places/temple.svg" alt="Temple" class="entity-icon large">
</div>
```
