Test Categories

1. Basic Search

Simple text searches across all fields.

Query: "Zeus"
Expected: Find Zeus deity with highlighted matches in title, summary, and content
Query: "creation"
Expected: Find creation myths across all mythologies
Query: "underworld"
Expected: Find underworld-related places, concepts, and deities

2. Fuzzy Matching (Typo Tolerance)

Test the system's ability to handle typos and suggest corrections.

Query: "Zues" (misspelled)
Expected: Still find Zeus with spelling suggestion "Did you mean: Zeus?"
Query: "Appollo" (misspelled)
Expected: Find Apollo with spelling correction

3. Boolean Operators

Complex queries using AND, OR, NOT operators.

Query: "thunder AND zeus"
Expected: Results must contain both "thunder" AND "zeus"
Query: "war OR battle"
Expected: Results containing either "war" OR "battle"
Query: "zeus NOT roman"
Expected: Results with "zeus" but excluding anything Roman
Query: "(war OR battle) AND greek NOT spartan"
Expected: Complex boolean combination

4. Exact Phrase Matching

Use quotes for exact phrase matches.

Query: "golden fleece"
Expected: Only results with exact phrase "golden fleece"
Query: "hero's journey"
Expected: Exact match for archetypal pattern

5. Wildcard Search

Use * for partial matching.

Query: "god*"
Expected: Matches god, goddess, gods, godly, etc.
Query: "zeu*"
Expected: Matches Zeus, Zeugma, Zeuxis, etc.

6. Field-Specific Search

Search within specific fields using field:value syntax.

Query: "mythology:greek"
Expected: Only Greek mythology content
Query: "contentType:deity"
Expected: Only deity entities
Query: "mythology:norse contentType:deity"
Expected: Only Norse deities

7. Faceted Filtering

Programmatic filter application.

Query: "war" with filters: mythology=greek, contentType=deity
Expected: Greek war deities only

8. Autocomplete Suggestions

Real-time suggestions as you type.

Input: "zeu"
Expected: Suggestions like Zeus, Zeugma, etc.

9. Search Analytics

View search statistics and trends.
