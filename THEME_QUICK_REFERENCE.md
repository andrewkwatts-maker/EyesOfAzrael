## Troubleshooting

**Themes not showing?**
- Clear browser cache
- Refresh page
- Check console for errors

**Shaders not animating?**
- Click the âœ¨ button to enable
- Check WebGL support (chrome://gpu)
- Try different quality setting

**Dropdown won't open?**
- Click the theme button again
- Check for JavaScript errors
- Disable browser extensions

## Browser Requirements
