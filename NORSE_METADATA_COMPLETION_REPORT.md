## Metadata Enhancements Added

### 1. Linguistic Metadata

Every entity now includes:

- **Original Old Norse Name** (with authentic characters: þ, ð, æ, ǫ, ý, etc.)
- **Old Norse Script** (originalScript field)
- **Pronunciation** (IPA notation)
- **Language Code** (`non` = Old Norse)
- **Etymology:**
  - Proto-Germanic or Old Norse root language
  - Semantic meaning and derivation
  - Cognates in 4-5 Germanic languages (English, German, Swedish, Icelandic, Norwegian)

**Examples:**
- Mjolnir: `Mjǫllnir` from Proto-Germanic *meldunjaz (crusher, lightning)
- Valhalla: `Valhǫll` from valr (the slain) + hǫll (hall)
- Yggdrasil: `Yggdrasill` from Yggr (Odin) + drasill (horse)
- Ragnarök: `Ragnarǫk` from ragna (gods) + rǫk (fate, twilight)

### 2. Geographical Metadata

Every entity now includes:

- **Origin Point** with precise Scandinavian coordinates
- **Latitude/Longitude** (regional accuracy for mythological concepts)
- **Modern Countries** (Norway, Sweden, Denmark, Iceland)
- **Regional Descriptions** (cultural/mythological context)

**Coordinate Examples:**

Every entity now includes:

- **Timeline Position:** Viking Age period classification
