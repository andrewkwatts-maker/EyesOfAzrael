Deity Domain Icons

Universal symbolic icons for deity domains across all mythologies

About Domain Icons

This icon set provides 15 symbolic, mythology-agnostic icons representing common deity domains found across world religions and mythologies. Each icon is designed to be instantly recognizable and works seamlessly across different cultural traditions.

Icons use currentColor to automatically adapt to mythology-specific color schemes, ensuring visual consistency while respecting each tradition's unique aesthetic.

Usage

Icons are automatically applied to deity pages based on their domains:

{
  "name": "Zeus",
  "type": "deity",
  "domains": ["sky", "thunder", "justice"],
  // Icon automatically selected from domains
}

See DEITY_DOMAIN_ICONS.md for complete documentation.
