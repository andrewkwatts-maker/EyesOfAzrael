## Icon HTML Structure

Each icon is injected into the hero section of entity pages:

```html
<section class="hero-section">
  <div class="hero-icon-display has-icon">
    <div class="entity-icon-container">
      <img src="/icons/deity-icon.svg"
           alt="Entity name icon"
           class="entity-icon hero-size"
           loading="lazy" />
    </div>
    <!-- Original emoji preserved as fallback -->
    ⚡
  </div>
  <h2>Entity Name</h2>
  ...
</section>
```

**Key Features:**
- CSS link added to `<head>` section
- Icons placed within `.hero-icon-display` div
- Original emoji preserved for browsers without SVG support
- `loading="lazy"` for performance optimization
- Semantic alt text for accessibility

## Maintenance Guide

### Adding New Icons

3. Run deployment script with `--execute`

### Updating Icons

1. Replace SVG file in `/icons/` directory
2. Clear browser cache

### Removing Icons

1. Delete icon reference from HTML
2. Remove icon file from `/icons/` directory
3. Update registry JSON
