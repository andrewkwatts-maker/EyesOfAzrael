Submission Link Component

A beautiful, reusable component for adding contextual submission buttons to any page.

Notice: The FAB (Floating Action Button) has been automatically injected in the bottom-right corner! Hover over it to see the label expand.

Features

  • Contextual awareness - automatically detects page type
  • Multiple button styles - FAB, inline, dropdown, card
  • Beautiful glass morphism design
  • Smooth animations and hover effects
  • Fully responsive and mobile-friendly
  • Accessibility support (ARIA labels, keyboard navigation)
  • Dark mode support
  • Auto-injection capability

Button Styles

1. FAB (Floating Action Button)

The floating action button appears in the bottom-right corner. It's been auto-injected on this page!

// Auto-inject FAB on page load SubmissionLink.autoInject(); // Or create manually const fab = new SubmissionLink({ style: 'fab' }); document.body.appendChild(fab.element);

2. Inline Button

Perfect for placement within page content or toolbars.

const button = SubmissionLink.createInline({ label: 'Add Deity', icon: '‚ûï' }); container.appendChild(button);

3. Dropdown Menu Item

Designed to fit seamlessly in navigation menus and dropdowns.

const menuItem = SubmissionLink.createDropdownItem({ label: 'Submit Content' }); dropdown.appendChild(menuItem);

4. Card Action Button

const cardButton = SubmissionLink.createCardButton({ label: 'Add Hero' }); card.appendChild(cardButton);

Context Detection

The component automatically detects the page context and provides appropriate labels:

// Automatically detects from URL: // /deities/ ‚Üí "‚ûï Add Deity" // /heroes/ ‚Üí "‚ûï Add Hero" // /creatures/ ‚Üí "‚ûï Add Creature" // /texts/ ‚Üí "‚ûï Add Text" // /teachings/ ‚Üí "‚ûï Add Teaching" // /theology/ ‚Üí "‚ûï Add Theology" // /theories/ ‚Üí "üìù Submit Theory" // Default ‚Üí "‚ûï Contribute" // Or override context manually: const button = new SubmissionLink({ context: { type: 'deity', label: 'Add Deity', icon: '‚ûï' } });

Usage Examples

Quick Auto-Injection

Add this to any page to automatically inject a FAB button:

<link rel="stylesheet" href="css/submission-link.css"> <script src="js/components/submission-link.js"></script> <script> SubmissionLink.autoInject(); </script>

Manual Placement

// Create and append to container const link = new SubmissionLink({ style: 'inline', label: 'Submit Your Theory', icon: 'üìù' }); container.appendChild(link.element); // Or use static render method for HTML string const html = SubmissionLink.render({ style: 'card', label: 'Add Creature' }); container.innerHTML = html;

Custom URL

const link = new SubmissionLink({ url: '/custom/submission/page.html', label: 'Custom Action' });

With Additional Classes

const link = new SubmissionLink({ className: 'my-custom-class another-class' });

API Reference

Constructor Options

new SubmissionLink({ style: 'fab', // 'fab', 'inline', 'dropdown', 'card' label: 'Custom Label', // Custom text (auto-detected if not provided) icon: '‚ûï', // Custom icon (auto-detected if not provided) context: null, // Force specific context (auto-detected if not provided) url: null, // Custom submission URL (calculated if not provided) className: '' // Additional CSS classes });

Static Methods

// Auto-inject FAB button SubmissionLink.autoInject(options); // Render HTML string const html = SubmissionLink.render(options); // Create specific button types const inline = SubmissionLink.createInline(options); const dropdown = SubmissionLink.createDropdownItem(options); const card = SubmissionLink.createCardButton(options); // Check if on submission page const isSubmitPage = SubmissionLink.isSubmissionPage(); // Remove all submission links SubmissionLink.removeAll(); // Update all submission links SubmissionLink.updateAll(options);
