## JSON Structure

Each mythology page follows this schema:

```json
{
  "id": "mythology_id",
  "type": "mythology_page",
  "name": "Mythology Name",
  "icon": "emoji",
  "tagline": "Brief catchphrase",
  "description": "One-sentence summary",
  "overview": "Comprehensive 200-300 word introduction",
  "culturalContext": {
    "period": "Time range",
    "region": "Geographic area",
    "language": "Primary languages",
    "primarySources": "Key texts"
  },
  "statistics": {
    "deities": number,
    "heroes": number,
    "creatures": number,
    // ... other entity counts
  },
  "sections": [
    {
      "id": "unique-section-id",
      "title": "Section Title",
      "type": "category-grid|content|list|parallels",
      "body": "Content text",
      "categories": [...], // for category-grid type
      "items": [...], // for list type
      "parallels": [...] // for parallels type
    }
  ],
  "relatedMythologies": ["id1", "id2", ...],
  "featured": true|false,
  "displayOrder": number,
  "metadata": {
    "createdAt": "timestamp",
    "updatedAt": "timestamp",
    "version": "1.0",
    "author": "EyesOfAzrael Content Team"
  }
}
```

## Section Types

### 1. Category Grid (`type: "category-grid"`)
Displays browsable entity categories with counts and routes:
```json
{
  "type": "category-grid",
  "categories": [
    {
      "type": "deities",
      "icon": "‚ö°",
      "label": "Deities",
      "count": 42,
      "route": "#/browse/deities/greek",
      "description": "Brief description"
    }
  ]
}
```

### 2. Content (`type: "content"`)
Rich text content with markdown support:
```json
{
  "type": "content",
  "body": "Narrative text with **bold**, *italic*, and multiple paragraphs..."
}
```

### 3. List (`type: "list"`)
Bulleted/numbered lists for sacred texts, sources, etc.:
```json
{
  "type": "list",
  "items": [
    "Item 1 - Description",
    "Item 2 - Description"
  ]
}
```

### 4. Parallels (`type: "parallels"`)
Cross-cultural connections:
```json
{
  "type": "parallels",
  "parallels": [
    {
      "mythology": "other_id",
      "connection": "Description of relationship",
      "strength": "primary|secondary|tertiary"
    }
  ]
}
```
