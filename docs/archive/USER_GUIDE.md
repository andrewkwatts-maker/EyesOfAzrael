## Searching for Content

### Quick Search

1. Click the **Search** link in the main navigation
2. Type your query in the search box
3. Results appear instantly as you type
4. Click any result to view the full entity page

### Search Modes

The search system offers multiple specialized modes:

#### Generic Search (Default)
- Searches across all fields: names, descriptions, domains, symbols
- **Example**: "thunder" finds Zeus, Thor, Indra, etc.

#### Language Search
- Finds entities by original language scripts
- **Example**: "ŒñŒµœçœÇ" finds Zeus in Greek script
- Searches transliterations and alternate spellings

#### Source Search
- Searches primary texts and archaeological sources
- **Example**: "Iliad" finds deities mentioned in Homer's Iliad
- Includes citations and scholarly references

#### Term Search
- Searches specialized corpus terms
- **Example**: "epithets:Sky Father" finds all sky father deities
- Searches domains, symbols, titles, and attributes

#### Advanced Search
- Combine multiple criteria with filters
- Filter by mythology, category, domains, symbols
- Sort by relevance, name, or date added

### Search Tips

- **Alternate Names**: Search "Enki" automatically includes "Ea" and "Nudimmud"
- **Cross-Cultural**: Search "Zeus" also finds Roman "Jupiter"
- **Wildcards**: Use partial names like "Apo*" to find Apollo, Apollon, etc.
- **Keyboard Shortcuts**:
  - `‚Üë` `‚Üì` arrows to navigate results
  - `Enter` to select highlighted result
  - `Esc` to close search

### Display Options

Switch between different result views:

- **Grid** - Visual cards (default, mobile-friendly)
- **Table** - Sortable columns with key data
- **List** - Compact list with expand/collapse
- **Panel** - Detailed cards with full descriptions

### Exporting Results

1. Perform your search

**Q: Is Eyes of Azrael free to use?**
A: Yes! Browsing and reading all content is completely free. Submitting theories requires a free Google account.

A: You may reference content for personal research and education. For commercial use or publication, please contact the copyright holder.

A: The site is continuously updated with new entities, theories, and features. Major updates occur monthly.

**Q: Where does the content come from?**
A: Content is compiled from public domain sources, scholarly research, primary texts, and user contributions (with attribution).

A: Google authentication provides secure, hassle-free sign-in without requiring you to create another password.

## Troubleshooting

### Common Issues

**Problem**: Site won't load or shows blank screen
- **Solution**: Clear browser cache, disable browser extensions, try incognito/private mode

**Problem**: Sign-in popup blocked
- **Solution**: Allow popups for Eyes of Azrael in browser settings, or use redirect sign-in

**Problem**: Images not displaying
- **Solution**: Check internet connection, disable ad blockers, clear browser cache

**Problem**: Shader backgrounds not animating
- **Solution**: Your browser may not support WebGL. Update your browser or use a different one.

**Problem**: Search not working
- **Solution**: Wait for page to fully load, refresh the page, clear browser cache

**Problem**: Theory submission fails
- **Solution**: Check internet connection, ensure you're signed in, reduce image sizes

### Browser-Specific Issues

**Chrome/Edge**:
- Clear cache: Ctrl+Shift+Delete (Windows) or Cmd+Shift+Delete (Mac)
- Hard reload: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)

**Firefox**:
- Clear cache: Ctrl+Shift+Delete (Windows) or Cmd+Shift+Delete (Mac)
- Hard reload: Ctrl+F5 (Windows) or Cmd+Shift+R (Mac)

**Safari**:
- Clear cache: Develop ‚Üí Empty Caches
- Hard reload: Cmd+Option+R
