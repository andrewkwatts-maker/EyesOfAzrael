Example 1: Yggdrasil - Norse World Tree

Type: Tree Diagram

The cosmic ash tree Yggdrasil connects the Nine Realms of Norse cosmology. This visualization shows the vertical structure with realms positioned in the crown, branches, trunk, and roots.

<div data-cosmology-diagram
data-type="tree"
data-mythology="norse"
data-structure="yggdrasil"></div>

Usage Instructions

Basic Setup

Include the required dependencies in your HTML:

<!-- D3.js -->
<script src="https://d3js.org/d3.v7.min.js"></script>

<!-- Cosmology Diagram CSS -->
<link rel="stylesheet" href="css/cosmology-diagrams.css">

<!-- Cosmology Diagram Renderer -->
<script defer src="js/components/cosmology-diagram-renderer.js"></script>

HTML Attributes

  • data-cosmology-diagram - Marks element as a diagram container (required)
  • data-type - Diagram type: "tree", "realm_map", "genealogy", or "cosmic_layers" (required)
  • data-mythology - Mythology name: "norse", "greek", "jewish", "egyptian", etc. (required)
  • data-structure - Structure name within the mythology (required)
  • data-entity-id - Firebase entity ID for dynamic loading (optional)

Available Structures

  • Norse: yggdrasil, nine_realms
  • Jewish: tree_of_life
  • Greek: underworld, olympian_genealogy
  • Egyptian: duat

Interaction Controls

  • Zoom: Mouse wheel or pinch gesture
  • Pan: Click and drag the diagram
  • Node Click: View detailed information
  • Reset View: Use the "Reset View" button
  • Export: Use the "Export PNG" button
  • Fullscreen: Use the "Fullscreen" button

Firebase Integration

To load cosmology data from Firebase, structure your Firestore data as:

entities/{mythology}/cosmology/{entityId}

Example:
entities/norse/cosmology/yggdrasil
