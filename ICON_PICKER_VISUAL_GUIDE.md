## Color Scheme

### Modal Background
- Background: `rgba(26, 31, 58, 0.95)`
- Backdrop Filter: `blur(20px)`
- Border: `2px solid rgba(139, 127, 255, 0.3)`

### Emoji Buttons
- Default: `rgba(0, 0, 0, 0.2)`
- Hover: `rgba(255, 255, 255, 0.1)` with purple border
- Selected: Golden gradient with glow shadow

### Action Buttons
- Clear: Transparent with border
- Select: Golden gradient
- Both have hover effects

## Integration Example

### In Panel Editor
```javascript
// User clicks icon picker button
pickIcon(index) {
  const panel = this.data.panels[index];

  // Create and show picker
  const picker = new IconPicker({
    currentIcon: panel.titleIcon || '',
    onSelect: (icon) => {
      panel.titleIcon = icon;
      this.refresh();
    }
  });

  picker.show();
}
```

### Modern Browsers (Full Support)
- Chrome 90+

- Safari 14+

### Fallback Behavior
- Backdrop filter: Solid background if unsupported
- LocalStorage: In-memory storage if unavailable

## Performance

### Optimization
- Lazy rendering of emoji grids
- Event delegation for emoji clicks
- Efficient DOM updates
- Small footprint (~14KB JS, ~12KB CSS)

### Memory Usage
- Single modal instance
- Reusable across multiple panels
- Automatic cleanup on destroy()
