## Badge Meanings

- **CI Pipeline**: Shows whether continuous integration tests are passing
- **Tests**: Shows whether the full test suite is passing
- **Deploy to Firebase**: Shows deployment status
- **codecov**: Shows code coverage percentage

## Installation

Copy the markdown above and paste it near the top of your README.md file, typically after the project title and description but before the main content.
