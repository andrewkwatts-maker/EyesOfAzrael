ACCESSIBILITY FEATURES
======================

ARIA ATTRIBUTES
- nav[aria-label="Breadcrumb"]: Semantic navigation
- aria-live="polite": Announces breadcrumb changes
- aria-atomic="false": Announces only changed items
- aria-current="page": Marks current breadcrumb
- aria-hidden="true": Hides decorative elements (icons, separators)

FOCUS STATES
- All links have 2px outline on focus-visible
- Outline-offset: 1px
- Color: var(--color-primary)
- Box-shadow provides additional contrast

CONTRAST RATIOS
- Primary links: 8.5:1 (WCAG AAA)
- Current item: 10:1 (WCAG AAA)
- Separators: 4.5:1 (WCAG AA)
- Hover states: Maintain or improve contrast

KEYBOARD NAVIGATION
- Tab: Forward through breadcrumbs
- Shift+Tab: Backward through breadcrumbs
- Enter: Navigate to breadcrumb link

SCREEN READER
- Reads breadcrumb trail naturally
- Current page announced via aria-current
- Icons/separators ignored (aria-hidden)
- Navigation landmark recognized

REDUCED MOTION
- Animations: disabled (animation: none)
- Transitions: disabled (transition: none)
- Transform effects: removed
- Opacity: set to final state immediately
