## Icon Design Standards

All 80 new icons adhere to strict design standards:

### Technical Specifications
```xml
ViewBox: 0 0 64 64
Stroke Width: 2px (2.5px for emphasis, 3px for bold elements)
Stroke Line Cap: round
Stroke Line Join: round
Color Scheme: currentColor (CSS themeable)
File Format: SVG (Scalable Vector Graphics)
File Size: <1KB each
```

### Design Principles
- **Simple line art style**: Clean, recognizable shapes
- **Consistent stroke weight**: 2-3px for visual harmony
- **No fills** (except strategic use of currentColor with opacity)
- **Scalable from 16px to 512px**: Maintains clarity at all sizes
- **Accessibility-friendly**: High contrast, clear shapes
- **Theme-compatible**: Uses currentColor for CSS theming

### Visual Style
- Minimalist geometric forms
- Culturally respectful representations
- Universal recognition priority
- Consistent visual language across all categories
