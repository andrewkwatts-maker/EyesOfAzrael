    <link rel="stylesheet" href="../../themes/theme-base.css">
    <link rel="stylesheet" href="../../themes/themes/day.css" id="theme-stylesheet">

THEME PICKER UI (Insert before </body> tag):

    <!-- Theme Picker UI -->
    <div id="theme-picker-container"></div>

COMMON CSS VARIABLE REPLACEMENTS:

Body styles:
    font-family: var(--font-primary);
    background: var(--color-background);
    color: var(--color-text-primary);
    line-height: var(--leading-normal);
    padding: var(--space-5);

Headers and sections:
    background: var(--color-surface);
    backdrop-filter: blur(10px);
    border: 2px solid var(--color-border);
    border-radius: var(--radius-xl);

Colors:
    color-primary -> var(--color-primary)
    color-secondary -> var(--color-secondary)
    text colors -> var(--color-text-primary) or var(--color-text-secondary)

Spacing:
    padding/margin values -> var(--space-X) where X is 1-20

Border radius:
    border-radius values -> var(--radius-sm/md/lg/xl/2xl/full)

Font sizes:
    font-size values -> var(--text-xs/sm/base/lg/xl/2xl/3xl/4xl/5xl)
