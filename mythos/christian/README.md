## Usage in EOAPlot System

These files are designed to integrate with the EOAPlot codex search system:
- **Inline Search Links**: Use `inline-search-link` class with `data-search-term` attribute
- **Expandable Sections**: Use `codex-search-section` class for collapsible content
- **JavaScript Functions**:
  - `toggleCodexSearch(header)` - Expand/collapse sections
  - `toggleVerse(citation)` - Show/hide verse text
  - `performCodexSearch(event, searchTerm)` - Integration point for search functionality
